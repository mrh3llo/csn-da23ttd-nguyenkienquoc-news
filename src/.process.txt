1. Code trong thư mục Code.
    Windows: D:\.SYSTEM\Documents\School\CSN\Code\
    Linux: ~/Documents/School/CSN/Code/

2. Chạy thử trên localhost (XAMPP).
    Trên Windows D:\APPS\Code\XAMPP\htdocs\kquoc\online_news\
    Linux /opt/lampp/htdocs/kquoc/online_news/

3. Nếu ổn định, từ thư mục Code, sao chép đến các thư mục khác (đồng bộ).
    Đối với Windows:
    Chạy từng lệnh:
        cp D:\.SYSTEM\Documents\School\CSN\Code\* D:\APPS\Code\XAMPP\kquoc\online_news\
        cp D:\.SYSTEM\Documents\School\CSN\Code\* D:\.SYSTEM\Documents\School\CSN\git\src\
    Chạy đồng thời 2 lệnh:
        cp D:\.SYSTEM\Documents\School\CSN\Code\* D:\APPS\Code\XAMPP\kquoc\online_news\ && cp D:\.SYSTEM\Documents\School\CSN\Code\* D:\.SYSTEM\Documents\School\CSN\git\src\

    Đối với Linux:
    Chạy từng lệnh:
        sudo cp -r ~/Documents/School/CSN/Code/* /opt/lampp/htdocs/kquoc/online_news/
        sudo cp -r ~/Documents/School/CSN/Code/* ~/Documents/School/CSN/git/src/
    Chạy đồng thời 2 lệnh:
        sudo cp -r ~/Documents/School/CSN/Code/* /opt/lampp/htdocs/kquoc/online_news/ && sudo cp -r ~/Documents/School/CSN/Code/* ~/Documents/School/CSN/git/src/

4. Push thư mục git lên github. Link tới repo: https://github.com/mrh3llo/csn-da23ttd-nguyenkienquoc-news.git
